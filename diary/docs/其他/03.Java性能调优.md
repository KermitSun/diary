## 常用建议

1. 尽量避免BigInteger和BigDecimal，金钱类可以使用int代替(金额乘以100);
2. Apache Commons StringUtils.Replace 代替String.replace;
3. 缓存昂贵的资源，比如数据库连接;
4. 减少对象的创建，比如SimpleDateFormat等线程不安全的对象，常常在方法内创建；
