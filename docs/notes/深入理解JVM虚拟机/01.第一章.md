## Java技术体系
- Java程序设计语言
- 各种硬件平台上的Java虚拟机
- Class文件格式
- Java API类库
- 来自商业机构和开源社区的第三方Java类库

jdk(Java Development Kit) : 包含Java程序设计语言、Java虚拟机、Java API类库;

jre(Java Runtime Environment) : 包含Java API类库中的SE API和Java虚拟机;

## Java虚拟机
