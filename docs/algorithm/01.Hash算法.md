# 1.散列表、散列函数
    散列表
        即为Hash表，英文名“Hash Table",只是翻译的不同释义；
        散列表使用数组，支持下标访问数据，时间复杂度O(1);
    散列函数:
        数据通过函数获得散列表的数组下标，这个函数称为散列函数；
        数据称为键或关键字，数组下标称为散列值；
        
# 2.常用散列函数
    MD5
    SHA
    CRC
    
# 3.散列函数设计的基本要求
    散列函数计算得到的散列值是个非负整数；
    如果key1 != key2，那hash(key1) != hash(key2);
